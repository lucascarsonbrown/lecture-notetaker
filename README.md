# Lecture Transcript

A minimal Chrome extension for UT Austin students that grabs the raw caption transcript from lecture videos and lets you copy it to your clipboard.

## Features

- One-click transcript extraction from UT lecture pages
- Copy to clipboard button with word count
- Dark mode

## Supported Sites

- `tower.la.utexas.edu`
- `lecturecapture.la.utexas.edu`

## Installation

1. Clone or download this repo
2. Go to `chrome://extensions` in Chrome
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** and select this folder

## Usage

1. Open a lecture video on one of the supported sites
2. Turn on **CC (closed captions)** and let the video play for a few seconds
3. Click the extension icon in your toolbar
4. Click **Get Transcript**
5. Click **Copy to Clipboard**
